# E-commerce app
## Objective
Make a fully functioning E-Commerce app.

## Clone the repository
```
git clone https://github.com/AAyushGuptAA/ShoppingD.git
cd ShoppingD
```

## Getting the Requirements
```
pip install -r requirements.txt
```

## Technologies used
- HTML, CSS, Bootstrap
- TypeScript
- Angular
- Ionic
- Django
- Mongo DB
